# TimeWise - Your AI-Powered Daily Planner

![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)

TimeWise is a smart, 100% client-side daily scheduling application that helps you organize your workday with the power of AI. Simply provide your task list, and let the AI assistant generate a prioritized and timed schedule for you.

This project is a unique collaboration between a human developer and an AI coding partner, with about 90% of the code being generated by AI from prompts.

## Key Features

- **AI Schedule Generation**: Automatically creates a detailed work plan from a simple list of tasks.
- **Dynamic Time Tracking**: See your remaining work time, overtime, and workday progress at a glance.
- **Interactive Schedule**: Easily edit tasks, change durations, and re-order your plan with drag-and-drop.
- **Break Tracking**: Log your breaks to get a more accurate picture of your workday.
- **EOD Reports**: Generate a professional end-of-day summary based on your completed tasks.
- **Client-Side Storage**: All your data is stored locally in your browser, ensuring privacy. No backend needed!
- **Light & Dark Mode**: Choose your preferred theme for a comfortable viewing experience.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: ShadCN UI
- **AI Integration**: Google AI with Genkit
- **State Management**: React Hooks (useState, useEffect, useLocalStorage)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1.  **Clone the repo:**
    ```sh
    git clone https://github.com/mitulparmar7161/time-wise.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd time-wise
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```
4.  **Get a Gemini API Key:**
    This project uses the Google Gemini API. You can get your free API key from [Google AI Studio](https://aistudio.google.com/apikey).

5.  **Run the development server:**
    ```sh
    npm run dev
    ```
    Open [http://localhost:9002](http://localhost:9002) with your browser to see the result. The app will prompt you to enter your API key on the first run.
