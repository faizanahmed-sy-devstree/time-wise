# **App Name**: TimeWise

## Core Features:

- Set Working Hours: Set daily working hours to define a standard workday duration.
- Arrival Time Input: Log arrival time to automatically start the work timer.
- Break Time Tracking: Record break durations to accurately calculate net work time.
- Real-Time Calculation: Dynamically display the remaining work time and predicted completion time.
- Countdown Timer: Display a prominent countdown timer showing time remaining, for increased clarity.
- API Key Encryption: Securely store the Gemini API key locally using encryption for AI feature access.
- AI Break Suggestion: Use AI as a tool to estimate optimal break times and suggest a break when user workload peaks are detected during the day based on calendar events (requires user authorization).

## Style Guidelines:

- Primary color: Slate blue (#778899) for a calm and professional feel.
- Background color: Light gray (#F0F8FF) for a clean, modern look.
- Accent color: Teal (#008080) to highlight important information and actions.
- Body font: 'Inter' (sans-serif) for a clean and readable interface.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and slightly techy feel. Use 'Inter' for body text, when longer text is required
- Use simple, outlined icons for clarity and ease of understanding.
- Ensure a responsive layout that adapts to different screen sizes for accessibility across devices.
